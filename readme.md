# Smart Router for IoT Device Security

INSE 6170 Course Project – Concordia University

This project implements a smart router application that allows monitoring and security management of IoT devices connected through a hotspot network.

## Features

- IoT device discovery
- Packet capture (PCAP)
- Whitelist-based firewall
- Intrusion detection
- Traffic logging and visualization

## Technologies

- Python
- Flask
- Scapy / PyShark
- SQLite
- Chart.js
